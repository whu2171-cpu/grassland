# Local Data Manifest

本文件记录大体积数据和本地结果路径。登记路径不代表文件进入 GitHub。

## Data Policy

- 原始遥感影像、`.tif`、`.npy`、模型权重和大型办公文档不提交到 Git。
- 每个数据条目应记录：数据名称、时间范围、空间范围、分辨率、本地路径、处理状态、依赖脚本、备注。
- 若后续使用 Git LFS 或 DVC，再单独建立追踪规则。
