## 当前模型结果

当前结果属于 proxy/internal validation。

| task | best model | OA | macro-F1 | features |
|---|---|---:|---:|---|
| fvc_ndvi | random_forest | 0.9989 | 0.9989 | mean_ndvi |
| subtype | hist_gradient_boosting | 0.6940 | 0.6947 | confidence, dem |

解释：

- `fvc_ndvi` 任务精度很高是预期现象，因为 label 由 `mean_ndvi` 分位数生成，模型输入也包含 `mean_ndvi`，该任务只用于验证链路可运行；
- `subtype` 任务当前只使用 `confidence` 和 `dem` 两个特征，精度不能代表最终模型能力；
- 后续需要加入 Sentinel-2 时序、NDVI/FVC 物候、ERA5、SoilGrids、坡度/TWI 等特征，并改为空间 block split。

## 边界说明

当前结果不能作为最终草地亚型独立验证精度。

当前结果可用于：

- 汇报模型链路最小可运行版本；
- 展示 proxy label 构建、模型训练、模型对比和精度表输出；
- 证明后续真实 reference samples 到位后，样本层可被替换。

当前结果不可用于：

- 声称草地亚型分类精度已经达到某一水平；
- 替代野外样方或权威草地类型图验证；
- 作为论文最终精度结果。

## 下一步

1. 增加环境因子栅格：ERA5、SoilGrids、slope、aspect、TWI；
2. 增加 Sentinel-2 或现有 NDVI/FVC 时序特征；
3. 改为空间 block split，降低空间自相关造成的精度虚高；
4. 获取或构建真实草地亚型 reference samples；
5. 实现正式 U-Net + Transformer 时空协同模型。
