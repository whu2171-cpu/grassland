# 2026-07-05 下一阶段研究任务书

## 目标

根据 `D:\草地遥感\中期\内蒙古草地遥感项目总结_A到O全方向.md` 和前期技术路线评估，下一阶段研究不再继续发散新的 FVC 驱动分析，而是优先补齐开题主线所需的“草原植被亚型分类证据链”。

核心目标：

> 以 Sentinel-2 时序影像和辅助环境因子为输入，构建“空间特征提取 + 时间特征提取 + 时空特征融合”的草原植被亚型高精度分类模型；ARNC/NDVI 物候阈值法只作为地面真值样本构建的辅助依据和轻量化 baseline，用于对比、解释和检验主模型，而不是替代主模型。

## 当前证据基础

### 已具备

- 2000-2024 年 FVC、温度、降水长时序数据；
- 260513 产出的 5 类草地亚型图、概率图、置信度图、GMM 状态图和 MK 趋势图；
- 0522/0528 的 FVC 趋势、RESTREND、GeoDetector、RF/SHAP、突变阈值、政策滞后和未来预测结果；
- 包头靶场多源高分辨率分类预实验；
- 徐雪峰论文提供的 ARNC 阈值法、过渡带类别、OA/UA/PA 精度评价和多期类型转移分析范式，可作为借鉴而非照搬；
- 开题技术路线中已经明确提出的时空特征协同提取框架：U-Net 编码器提取空间纹理/斑块特征，Transformer Encoder 提取时序物候变化特征，并通过 Concat/Add 等方式进行特征融合。

### 主要缺口

- 目前高 R2 结果主要来自 FVC 连续变量预测，不能替代亚型分类精度；
- 缺少围绕 U-Net + Transformer 的主模型设计、训练策略、消融实验和与 baseline 的系统对比；
- 缺少本研究区的 ARNC baseline 和过渡带分类结果，但 ARNC 只能作为 baseline 和样本辅助，不能成为最终技术路线主体；
- 缺少独立验证集、混淆矩阵、OA、UA、PA、macro-F1；
- 缺少多期亚型转移矩阵和桑基图；
- 现有驱动分析目标变量多为 FVC/NDVI，而不是亚型转移概率或退化等级。

## 研究路线

### 阶段 1：样本与数据盘点

任务：

1. 盘点 260513 输出的亚型分类、概率、置信度、GMM 状态和趋势产品；
2. 盘点可作为验证的 CSV、GeoJSON、野外样点和 GEE 采样点；
3. 明确训练样本、验证样本和仅作辅助解释的数据边界；
4. 建立 `data_index/classification_sample_inventory.md`。

验收标准：

- 列出每个样本/产品的路径、类别体系、空间范围、年份、分辨率、是否可用于独立验证；
- 明确哪些数据不能直接作为独立验证。

### 阶段 2：ARNC baseline 与地面真值样本辅助

任务：

1. 基于现有 NDVI/FVC 时序计算年内或跨年 ARNC 指标；
2. 按草甸草原、典型草原、荒漠草原、草原化荒漠/荒漠等类别统计 ARNC 分布；
3. 设计 5 类硬分类和相邻过渡带分类方案；
4. 将 ARNC 结果用于三类目的：辅助筛选高可信样本、提供轻量 baseline、解释相邻亚型过渡带；
5. 输出阈值表、分类图、小图和精度表。

验收标准：

- `results/tables/arnc_thresholds.csv`
- `results/tables/arnc_accuracy.csv`
- 混淆矩阵、OA、UA、PA、macro-F1；
- 记录至少一版失败或不理想的阈值实验，避免只保留最好结果；
- 明确写出 ARNC 的边界：它不是最终高精度模型，只用于 baseline、样本辅助和过渡带解释。

### 阶段 3：时空特征协同提取主模型

任务：

1. 构建数据输入与样本结构：
   - Sentinel-2 多时相影像作为主输入；
   - DEM、坡度、气候、土壤和人类活动代理变量作为辅助环境因子；
   - 野外样方、人工解译点、历史调查图和 ARNC 辅助筛选结果共同构建训练/验证样本；
   - 对所有输入进行数据标准化、时空对齐和掩膜处理。
2. 构建空间特征提取分支：
   - 使用 U-Net 编码器提取空间纹理、斑块边界和局部上下文；
   - 输出可与时序分支融合的多尺度空间特征。
3. 构建时间特征提取分支：
   - 使用 Transformer Encoder 或轻量时序注意力模块提取多时相 Sentinel-2 植被指数/光谱变化；
   - 重点捕捉返青、峰值、生长季长度和衰退期等物候差异。
4. 构建时空特征融合层：
   - 比较 Concat、Add 或注意力融合方式；
   - 形成统一的像元/斑块级亚型分类特征。
5. 进行模型分阶段训练与优化：
   - 先训练轻量可运行版本；
   - 再加入环境因子和多尺度空间特征；
   - 最后进行主模型、ARNC baseline 和传统机器学习模型的统一对比。
6. 输出高精度植被亚型分类结果和不确定性图。

验收标准：

- 主模型结构图或模块说明：U-Net 空间分支、Transformer 时间分支、融合层、分类头；
- 至少一版可运行训练脚本和配置文件；
- 同一验证集上的模型对比表：ARNC、DT/RF/XGBoost、U-Net、Transformer-only、U-Net + Transformer；
- 每类 OA、PA、UA、F1、macro-F1；
- 消融实验：无环境因子、无时间分支、无空间分支、不同融合方式；
- 至少一次空间外推验证或空间分块交叉验证，避免随机像元划分导致精度虚高。

### 阶段 4：多源机器学习与传统模型对照

任务：

1. 在同一训练/验证样本上比较 DT、RF、XGBoost 或轻量模型；
2. 输入特征包括 FVC/NDVI 物候、降水、温度、DEM、坡度、土壤或人类活动代理变量；
3. 与 ARNC baseline 和时空协同主模型比较精度、稳定性和空间一致性；
4. 输出误分类空间图和特征重要性。

验收标准：

- 同一验证集上的传统模型对比表；
- 每类 PA/UA/F1；
- 传统模型与深度时空模型的误分类区域对照；
- 明确哪些特征只用于传统模型解释，哪些进入深度模型输入。

### 阶段 5：亚型变化与驱动回接

任务：

1. 基于主模型输出的多期高精度亚型图生成转移矩阵和桑基图；
2. 定义亚型退化等级或转移强度；
3. 以亚型变化强度为目标变量，重跑或改写 GeoDetector/SHAP/阈值分析；
4. 将 FVC 突变阈值作为生态响应解释，而不是分类结果本身。

验收标准：

- 多期亚型转移矩阵；
- 退化方向图：草甸草原 -> 典型草原 -> 荒漠草原 -> 草原化荒漠/荒漠；
- 驱动因子解释结果表；
- 明确哪些结论属于“关联解释”，哪些可以谨慎讨论为“政策响应”。

## 技术路线定位修正

参考论文的作用是借鉴三个环节，而不是复制其完整路线：

1. 借鉴 ARNC 作为可解释、轻量化的物候 baseline；
2. 借鉴过渡带思想，用于解释相邻草地亚型混淆和不确定区；
3. 借鉴 OA/UA/PA、混淆矩阵和野外样方验证的精度评价方式。

本研究真正需要突出的创新路线是：

> 多源数据输入与样本构建 -> Sentinel-2 时序影像和环境因子时空对齐 -> U-Net 空间特征提取 -> Transformer 时间特征提取 -> 时空特征融合 -> 高精度植被亚型分类 -> 交叉验证与不确定性评估 -> 多期亚型变化检测 -> GeoDetector/SEM 等驱动机制解释。

因此，ARNC、DT、RF、XGBoost 都是对照或辅助模块；主模型应围绕“时空特征协同提取”展开。
