# 草地亚型地面真值与验证样本方案

日期：2026-07-06

## 1. 参考论文中的地面真值来源

徐学锋论文并不是把模型输出或聚类结果直接当作地面真值，而是分成了三个层次：

1. 历史草地类型调查图/植被类型图作为参考数据和先验知识；
2. 清洗后的历史调查图像元作为模型训练和内部测试样本；
3. 2021 年实地样方作为外部野外验证。

其中，草地类型分布调查数据包括：

- 1980s 中亚五国植被类型图，基于原苏联纸质地图数字化，并由实测资料补充；
- 1980s 中国植被类型图，该图以野外实地调绘和生物产量测定为主、遥感影像校核为辅；
- 2009 年内蒙古草地类型调查产品，该数据以遥感影像为背景，叠加野外调查数据、1980s 草地类型图和 2001-2003 年草原亚类图后综合判读勾绘；
- 蒙古国 1990 年和 2010 年草地覆盖数据，基于 Landsat 影像解译和面向对象分类。

对 ARNC 方法，论文将 2000s ARNC 图与内蒙古温性草地类型图叠加，统计不同草地类型对应的 ARNC 阈值范围。由于历史调查图仍可能包含人工误分，作者使用 3-sigma 思路进行数据清洗，最终采用均值 ± 1.5 倍标准差范围保留样本。

对机器学习方法，论文以清洗后的 1980s 和 2000s 像元作为样本，随机选取 70% 作为训练集、30% 作为测试集。该测试属于基于同源参考图的内部精度评价，不等同于完全独立野外验证。

真正的外部野外验证来自 2021 年 7-8 月内蒙古东部草地野外调查：

- 共 143 个样方；
- 采样区覆盖 13 个旗县；
- 记录经纬度、物种组成、植被覆盖度和生长状况；
- 根据样方记录为样本点添加草地类型属性；
- 用于验证 2010s 分类结果。

该野外验证中，ARNC 方法在非过渡带 41 个样方中错 3 个，正确率 92.7%；在过渡带 102 个样方中错 3 个，正确率 97%。DNN 和 DT 的 2010s 预测结果也用同一批 143 个样方验证，DT 正确 126 个、正确率 88%；DNN 正确 104 个、正确率 72.7%。

## 2. 其他草地/土地覆盖分类研究的常见做法

从本地参考论文和土地覆盖精度评估规范看，草地亚型或草地群落分类常见地面真值来源有四类：

### 2.1 野外样方/样点

这是最强的地面真值。常见记录内容包括：

- 经纬度；
- 优势种或物种组成；
- 群落盖度、高度、生物量；
- 生境条件，如土壤水分、土壤质地、坡度、海拔；
- 现场照片；
- 专家判定的草地类型或群落类型。

例如，本地“内蒙古草原分类”相关论文使用 164 个内蒙古草地样点，每个样点内设置多个样方，调查植被和土壤属性，再基于群落状态划分类型。

### 2.2 高分辨率影像人工解译

当无法获取足够野外样本时，可用 Google Earth、高分影像、无人机影像或 Sentinel-2/Landsat 时序影像进行人工解译。适合识别明显地类或植被斑块，但对草地亚型要谨慎，因为草甸草原、典型草原、荒漠草原之间往往不是单靠影像纹理就能稳定区分。

### 2.3 权威历史图件/调查产品

草地资源调查图、植被类型图、草原亚类图可以作为训练先验或参考图。但这类数据通常存在尺度差异、年代差异和制图误差，不能无条件称为“地面真值”。更稳妥的表述是 reference map 或 historical survey-based reference data。

### 2.4 模型高置信伪标签

用已有分类产品和置信度图筛选高置信样本，可以作为第一版模型训练样本，但只能称为 pseudo-label。它不能作为独立验证，因为训练标签和验证标签同源，会高估精度。

## 3. 对本研究的直接判断

当前 `Hulunbuir_Labeled_Points.csv` 不能直接作为草地亚型地面真值：

- `landcover` 全部为 30，更可能是草地掩膜或土地覆盖编码；
- `Y_cluster` 是 0-3 的聚类/生态状态标签，未证明对应草地亚型；
- 该文件没有明确独立野外亚型标签来源。

因此，本研究第一阶段应采用双轨策略：

1. ARNC baseline 训练：使用 `Subtype_Classification_Full.tif` + `Subtype_Confidence_Full.tif` 构建高置信 pseudo-label 样本；
2. 独立验证建设：从野外样方、人工解译点、历史草地类型图或调查产品中建立真正 reference samples。

## 4. 建议的本研究样本分级

| 等级 | 样本来源 | 可用于训练 | 可用于独立验证 | 推荐表述 |
|---|---|---:|---:|---|
| A | 野外样方/样点，有优势种和草地亚型 | 是 | 是 | field reference samples |
| B | 专家基于高分影像和生态背景人工解译 | 是 | 可谨慎使用 | visually interpreted reference samples |
| C | 权威历史草地类型图/调查产品 | 是 | 可作参考验证，但需说明尺度和年代差异 | survey-based reference map |
| D | 现有分类图高置信像元 | 是 | 否 | pseudo-label training samples |
| E | 聚类结果或生态状态标签 | 可辅助解释 | 否 | ecological state / cluster labels |

## 5. 下一步操作建议

1. 查找开题材料和已有数据中是否存在真实野外样方表，重点字段为经纬度、优势种、盖度、草地类型、调查日期；
2. 若没有可用野外样方，先建立 100-200 个高置信人工解译点，覆盖 class 0-4 和过渡带；
3. ARNC baseline 先用 D 级 pseudo-label 训练，并用 spatial block split 做内部一致性验证；
4. 论文精度结论只使用 A/B/C 中经过人工复核的样本；
5. 报告指标至少包括 confusion matrix、OA、UA、PA、macro-F1；若样本数允许，补充类别不平衡情况下的 balanced accuracy。
